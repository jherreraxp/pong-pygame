# Pong Pygame

A pong game powered by [Pygame][1].

## Installation

Install `SDL`, `smpeg` and `portmidi`:

```
>> macOS
$ brew install sdl sdl_image sdl_mixer sdl_ttf smpeg portmidi
```

Then, install Python dependencies:

```
$ pip install -r requirements.txt
```

## Usage

Run the main script:

```
$ python src/main.py
```
